# Internal RSS Streams

## Original Author: Leo Benkel

## Status: BRAINSTORMING

## Idea

### Description

A platform for enterprise. An internal tool.
Each team/people can post update with different importance.
This will allow teams and people to learn about each other. 
Know what is going on.

## Use cases

* priority 0    : We are changing the schema of this database table
* priority 2    : We are updating our library to version 1.2, please udpate your project.
* no priority   : We are hosting a happy hour tomorrow.
* priority news : We are starting the work on project AAA
* priority 0    : Office will be closed tomorrow for fire alarm testing
